PPE AFC

Introduction

Lors de ma deuxième année de BTS SIO au lycée Victor Hugo, j'ai du effectué un deuxième PPE (projet personnel encadré) sur la continuité du PPE précédent. Ce PPE nommé AFC (APPLI-FRAIS-COMPTABLE) est à développé pour l'entreprise GSB qui a pour but d'enregistrer des frais engagés et permet le suivi des remboursement.

Ce projet est divisé en 3 parties:
La première partie consiste a mettre en place la procédure de saisie et des frais de remboursement
La deuxième partie consiste à réaliser des statistiques sur la base données à partir d'une page proposant plusieurs requêtes à l'utilisateur
La troisième partie consiste à développer le module destiné aux comptable pour valider et suivre le remboursement des frais de déplacement enregistrés par les visiteurs

Objectifs

Développer un site WEB dynamique pour l'entreprise GSB, ce projet aura une architecture MVC (Model View Controller) et sera donc lié à une base de données sur SQL server.

Outils Utilisés

  • Microsoft SQL Server est un système de gestion de base de données (SGBD) en langage SQL, et développé par Microsoft

  • Swift est un langage de programmation objet, principalement utilisé pour le développement d'application pour l'ecoSystème d'Apple

  • GitHub est une plateforme d'hébergement de code pour le contrôle de version et la collaboration. Elle permet, ainsi qu'à d'autres personnes, de travailler ensemble sur des projets, où que vous soyez, Github propose également un service de question réponses sur le développement informatique

  • Stack Overflow est un site web proposant des questions et réponses sur un large choix de thèmes concernant la programmation informatique.

Apport du projet

Ce projet m'a permis de :
- Utiliser l'architecture MVC.
- Travailler en équipe
- Développer un site WEB dynamique

Conclusion

Ce projet m'a permis de m'améliorer dans le développement WEB en gérant la partie client et serveur. Il m'a également appris a suivre un cahier des charges bien précis afin de respecter les attentes du client
