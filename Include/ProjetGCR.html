PPE GCR

Introduction

Lors de ma première année de BTS SIO au lycée Victor Hugo, j'ai du effectué un PPE (projet personnel encadré). Ce PPE nommé GCR (GESTION COMPTES-RENDUS) est à développé pour l'entreprise GSB (entreprise pharmaceutique) qui a pour but, comme son nom l'indique, de gérer les comptes rendu des visites et de production d'éléments statistiques

Objectifs

Développer un site WEB dynamique pour l'entreprise GSB permettant l'enregistrement des rapports de visite, ce projet sera donc lié à une base de données sur mySQL.

Outils Utilisés

  • Ordinateur sous Windows 10

  • Netbeans est un IDE(environnement de développement intégré) placé en open source, il est utilisé principalement pour le développement WEB

  • MySQL est un système de gestion de bases de données relationnelles.

  • GitHub est une plateforme d'hébergement de code pour le contrôle de version et la collaboration. Elle permet, ainsi qu'à d'autres personnes, de travailler ensemble sur des projets, où que vous soyez, Github propose également un service de question réponses sur le développement informatique

  • Stack Overflow est un site web proposant des questions et réponses sur un large choix de thèmes concernant la programmation informatique.
